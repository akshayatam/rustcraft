# RustCraft 🦀 - A Rust Learning Repository
Welcome to my Rust learning repository. This project documents my journey of learning Rust through structured practice, hands on exercises, and detailed notes.

The goal of this repository is not just to learn Rust, but to build a deep understanding of systems programming concepts, ownership, and performance oriented design.

---

## 📌 What this repository contains
### 1. Chapter Practice

This folder contains code written while working through the book [The Rust Programming Language](https://doc.rust-lang.org/book/)

Each chapter has its own directory.

Example structure:

```bash 
ch02/
  guessing_game.rs
ch03/
  variables.rs
  control_flow.rs
```

These files represent experiments, exercises, and variations beyond the book examples.

### 2. Rustlings Practice

This folder tracks my progress using [rustlings](https://rustlings.rust-lang.org/)

Rustlings is a set of small exercises designed to teach Rust concepts interactively.

The folder is kept in sync across multiple machines to ensure consistent progress.

### 3. Obsidian Notes

This folder contains structured notes created using Obsidian

The notes are organized chapter wise and include:

- Key concepts
- Code explanations
- Mental models
- Common mistakes
- Personal insights

The goal is to convert passive reading into active understanding.

---

## 🎯 Learning Approach

This repository follows a layered approach:

1. Read concepts from the book
2. Implement and experiment with code
3. Reinforce learning through Rustlings
4. Write structured notes to solidify understanding

This cycle helps in long term retention and practical mastery.
